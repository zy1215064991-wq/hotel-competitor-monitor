配置顺序

FlyAI 只负责飞猪生态价格。Key 只放 Windows 环境变量，项目文件里不要写明文。

1

获取 FlyAI Key

打开官方入口，申请酒店搜索能力。

https://flyai.open.fliggy.com/#ability
2

写入 Windows 环境变量

setx FLYAI_API_KEY "替换成你的 FlyAI Key"
3

安装 FlyAI CLI

npm i -g @fly-ai/flyai-cli --registry=https://registry.npmmirror.com
4

本地自检

Get-Command flyai

重新打开 WorkBuddy 或 PowerShell 后再检查，避免旧进程读不到新环境变量。

运行边界

DryRun 不调用 FlyAI。正式运行后，报告里的 FlyAI Usage 会显示真实调用次数、空结果、失败和脱敏价格数量。
