WorkBuddy 本地部署包

酒店竞对监控配置向导

0/4 已完成

浏览器与登录

先确认 WorkBuddy 能用真实 Chrome 或 Edge 浏览器打开携程，并完成一次扫码登录。

酒店识别

输入酒店名或关键词，让 WorkBuddy 用真实浏览器去携程搜候选。用户只确认候选，不手填页面链接。

默认 3 家，可按门店实际监控范围调整。

粘贴 WorkBuddy 返回的候选表

建议让 WorkBuddy 输出 Markdown 表格，列为：角色、候选序号、酒店名、地址、评分、hotelId、稳定链接、匹配理由。

默认查询口径

这些是每天定时任务的默认值。临时查询可以在 WorkBuddy 对话框里用自然语言覆盖。

生成配置

确认候选酒店后，导出 WorkBuddy 需要的配置文件和提示词。
